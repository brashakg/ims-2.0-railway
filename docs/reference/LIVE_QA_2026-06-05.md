## Severity legend
- **P0 Blocker** — data loss, security hole, page crash, money/GST wrong, cannot complete core flow
- **P1 Critical** — feature broken, wrong result, RBAC leak, no error on failure
- **P2 Major** — partial break, bad UX on a real path, missing validation
- **P3 Minor** — cosmetic, contrast, copy, slow-but-works, empty-state polish
