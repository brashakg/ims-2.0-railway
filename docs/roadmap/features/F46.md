2. **Consent gate wins; `is_transactional=True` bypasses:** A customer with `marketing_consent=False` (or most-recent `whatsapp_consent_ledger` event = `OPT_OUT`) receives `skipped_consent=1` from every rule where `is_transactional=False`. Assert the call goes through `marketing.is_opted_out` at marketing.py:1463 — not a local re-implementation (import the function; test it is the gate).
